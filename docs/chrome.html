Como funciona

O Polypus CLI (que você já tem ou instala agora) se conecta via WebSocket com a extensão. O agente "vê" a página, decide ações, e a extensão as executa no navegador.

Instalação

  1. Instale o CLI Polypus 
    npm install -g @gaberrb/polypus
  2. Inicie o servidor WebSocket 
    polypus web-server --port 9876

    Deixe rodando em segundo plano. Use um terminal separado ou um gerenciador de processos.

  3. Instale a extensão

    Carregue a extensão no Chrome:

    1. Abra chrome://extensions
    2. Ative "Modo do desenvolvedor"
    3. Clique "Carregar sem compactação" e selecione a pasta apps/chrome/
  4. Conecte e use!

    Abra uma página, clique no 🐙 na barra de ferramentas e digite sua tarefa.

⚡ A extensão se conecta automaticamente ao WebSocket. O status "🟢 Conectado" aparece no popup quando tudo está funcionando.

Modos de permissão

📖 Plan

Somente leitura. O agente navega e extrai conteúdo, mas nunca clica, digita ou modifica nada. Ideal para pesquisas.

🔍 Review padrão

Cada ação do agente (clicar, digitar, navegar) é exibida no side panel para você aprovar ou rejeitar antes de executar.

⚡ Bypass

Ações são executadas automaticamente. Use com allow-list de domínios para segurança.

Guias de uso avançado

📋 Criar uma issue no GitHub

1. Configure permissão: modo review, allow-list com github.com.
2. Navegue até o repositório: "Vá para github.com/GaberRB/polypus".
3. Peça: "Crie uma issue com título 'Bug no parse de XML' e corpo 'Quando o modelo devolve XML malformado, o parser falha silenciosamente'".
4. O agente clica em "New Issue", preenche título e corpo, e submete. Cada ação aparece para aprovação.

📊 Extrair dados estruturados

1. Navegue até a página com os dados.
2. Peça: "Extraia todos os produtos desta página e salve num arquivo CSV com as colunas: nome, preço, link".
3. O agente extrai o texto, estrutura os dados e chama write_file para salvar.
4. O CSV estará no diretório onde o polypus web-server foi iniciado.

🧪 Preencher formulário automaticamente

1. Permita o domínio do site na allow-list.
2. Modo review (padrão) para conferir cada campo.
3. Peça: "Preencha o formulário de cadastro com nome 'Teste', email 'teste@example.com', e selecione a opção 'Plano Pro'".
4. O agente preenche campo por campo. Você aprova antes de cada envio.

FAQ

Preciso pagar alguma assinatura?

Não. Você usa sua própria chave de API. A extensão é open source e gratuita. Você paga apenas o consumo dos modelos de IA que escolher.

A extensão consegue ver minhas senhas?

A extensão só vê o que está visível na página. No modo review, cada ação precisa da sua aprovação. Em modo plan, ela não clica nem digita. O código é open source e auditável.

Funciona com qualquer modelo de IA?

Sim. Funciona com qualquer modelo via OpenRouter, Ollama (local) ou API compatível com OpenAI. Modelos com visão podem usar screenshots como contexto.

Os dados saem da minha máquina?

Apenas as chamadas para o provedor de IA que você configurou. O WebSocket entre a extensão e o CLI é local. Nenhum dado é enviado para servidores do Polypus.

Como atualizar a extensão?

Puxe as mudanças do repositório (git pull), rode npm run build em apps/chrome/, e clique em "Atualizar" em chrome://extensions.

Solução de problemas

WebSocket não conecta

Verifique: (1) O polypus web-server está rodando? (2) A porta 9876 está livre? (3) Algum firewall está bloqueando conexões locais?

Extensão não carrega

Confirme que está em "Modo do desenvolvedor" em chrome://extensions. Selecione a pasta apps/chrome/ inteira. Se houver erro de manifest, leia a mensagem.

Ação web não executa

Tente "extraia todos os elementos interativos" para ver seletores disponíveis. Confirme que a página carregou. Use modo review para ver a ação exata.

Arquitetura

A extensão Chrome se comunica com o CLI Polypus via WebSocket local (porta 9876). O service worker gerencia a conexão, o side panel exibe o chat com streaming, e o content script executa as ações (navegar, extrair, clicar, digitar) na página ativa.

🔒 Tudo roda localmente. Nenhum dado sai da sua máquina além da chamada ao provedor de IA que você escolheu (OpenRouter, OpenAI, Anthropic, Ollama…). A extensão não exfiltra informações das páginas que você visita.
